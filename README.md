# Barcode Generator
Generates a barcode based upon a given input string.

## Compiling
Requires JDK 21!
### MacOS/Linux
Open Terminal and run
```shell
./gradlew build
```
### Windows
Open Command Prompt and run
```
./gradlew.bat build
```

You will find the compiled at /build/libs/BarcodeGenerator-1.0-all.jar
